consulate
=========
Consulate is a Python client library and set of application for the Consul
service discovery and configuration system.

|Version| |Downloads| |License|

Installation
------------
consulate may be installed via the Python package index with the tool of
your choice. I prefer pip:

.. code:: bash

    pip install consulate

Requirements
------------
- requests
- simplejson
- tornado (optional)
